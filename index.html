【Markdown】Typora画流程图实践

发表于 2021-01-08 更新于 2021-01-09 分类于 技术笔记 ， markdown Valine：

Typrora中使用了Mermaid来提供强劲的绘制流程图、时序图的功能。

关于Mermaid

是一种简单的类似 Markdown 的脚本语言，通过 JavaScript 编程语言，将文本转换为图片。因此，真正实现画图功能的并不是 Typora 本身，它只是内置了对 Mermaid 的支持。

图形介绍

流程图(flowchart)

主要语法说明：

graph - 表明是流程图

方向：

  • TB - top to bottom
  • TD - top-down/ same as top to bottom
  • BT - bottom to top
  • RL - right to left
  • LR - left to right

时序图

语法解释：->> 代表实线箭头，-->> 则代表虚线。

tqdm进度条的使用

发表于 2020-12-23 分类于 技术笔记 ， python Valine：

以前使用的是processbar来制作进度条， 听闻tqdm性能更好些，遂尝试使用一下。

参数使用：

ascii = True, 缺省是False, 当使用它时，不是以正常块状来做进度条，而是以"123456789#"的方式。

ncols = 90, 这里可以根据屏幕大小来进行控制，保证进度条不产生换行的情况。

colour = "green", 在版本4.5之后引入，可以设置进度条的颜色。

desc或利用set_desciption，用来设置进度条左侧的文字信息。

从使用情况来看，还是比较令人满意的。
