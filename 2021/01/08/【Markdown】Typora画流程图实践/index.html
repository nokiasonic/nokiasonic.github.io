【Markdown】Typora画流程图实践

发表于 2021-01-08 更新于 2022-04-14 分类于 技术笔记 ， markdown 阅读次数： Valine：

Typrora中使用了Mermaid来提供强劲的绘制流程图、时序图的功能。

关于Mermaid

是一种简单的类似 Markdown 的脚本语言，通过 JavaScript 编程语言，将文本转换为图片。因此，真正实现画图功能的并不是 Typora 本身，它只是内置了对 Mermaid 的支持。

图形介绍

流程图(flowchart)

主要语法说明：

graph - 表明是流程图

方向：

  • TB - top to bottom
  • TD - top-down/ same as top to bottom
  • BT - bottom to top
  • RL - right to left
  • LR - left to right

时序图

语法解释：->> 代表实线箭头，-->> 则代表虚线。
